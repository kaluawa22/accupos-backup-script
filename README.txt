Deploying Script using AccuServer Database and Task Scheduler 

1. Make sure you create a .bat file for the script
2. Run whether user is logged on or not 
3. Run with the highest privileges
4. Under actions make sure you've added the .bat file location under 'Program/script' i.e., C:\ScheduledJobs\ISBNSales\run_script.bat
5. In the 'Start in' field be sure to put the location of the .bat filed i.e, C:\ScheduledJobs\ISBNSales\
